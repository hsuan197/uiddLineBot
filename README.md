# Linebot
  * install
    * `yarn`
  * make your config.json
    * `cp config.json.sample config.json`
  * edit your config.json
    * port / password / channelId / channelSecret / channelAccessToken
  * run 
    * `node lineser.js`
  * sign in https://developers.line.biz/en/
  * change Webhook URL
